## Old SEAD Club's website

## Installation

```sh
git clone https://github.com/seadclub/sead.club.js
```

- Install all dependencies

```sh
yarn install
```

## Usage

- Run

```sh
yarn start
```

- Run in dev mode

```sh
yarn dev
```
